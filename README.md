# 🤖 HR Management Website with Integrated AI Capabilities

## 📌 Description
This project is a web-based Human Resource Management System developed as a university project.  
It aims to simplify and automate HR operations such as employee management, attendance tracking, leave requests, and performance evaluation.

---

## 🎯 Features
- Employee Management System  
- Attendance Tracking (Check-in / Check-out)  
- Leave Request and Approval System  
- Performance Evaluation  
- Role-Based Access (HR / Employee / Manager)  
- Secure Login System  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- PHP  
- MySQL  

---

## 📂 System Modules
- Employee Dashboard  
- HR Dashboard  
- Leave Management  
- Attendance System  
- Document Management  
- Messaging System
